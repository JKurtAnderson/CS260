Getting Started

Review this page for instructions on how to get started on the projests.

Starter Code

We've created some IntelliJ projects that you can use to start programming the labs. Follow these instructions to use these template projects on your own.

  1. Hover over the "Getting Started" tab in the menu and select the lab that you'd like to start. This will take you to that project's GitHub page.
  2. Click "Clone or download" and choose how you'd like to download the project. These steps will assume that you select "Download ZIP" but if you know how clone a repository on your own, feel free to do that.
  3. If you downloaded as a ZIP in the last step, extract the contents of that archive to the directory that you'd like your project to reside in.
  4. Open IntelliJ. Select File->Open and navigate to the directory in which you placed the extracted project and select it.
  5. You're ready to go! Good luck.
